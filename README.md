# Racket-task
Operates a task for data collection using an accelerometer fixed to the center of a tennis table racket. 
The data is acquired through a parallel cable plugged into a LabJack DAQ.

The task consists of moving the racket so that its velocity moves a static ball in the screen to a target line. 
Participants can self-control the amount of trials to practice. 
The software can also handle the control group (yoked).
